# 矫正胎龄计算器
一个轻量、无依赖的静态网页工具，支持通过**出生日期、简化格式的出生胎龄、目标日期**，快速计算目标日期对应的矫正胎龄。


## 工具功能
- 支持简化的胎龄输入格式：`34`（对应34周）、`34+2`（对应34周2天）
- 自动计算“出生日期→目标日期”的间隔天数
- 实时生成目标日期的矫正胎龄（格式：`XXW` 或 `XXW+X`）
- 内置表单验证（空值、格式错误、日期顺序错误）
- 纯静态网页，可离线使用，也可部署到线上


## 使用方法
1. **保存文件**：将工具代码保存为`index.html`
2. **打开网页**：双击`index.html`，在浏览器中直接打开（支持Chrome、Edge、Firefox等现代浏览器）
3. **输入信息**：
   - 出生日期：选择或输入`YYYY-MM-DD`格式的日期
   - 出生胎龄：输入`34`（对应34周）或`34+2`（对应34周2天）
   - 目标日期：选择或输入`YYYY-MM-DD`格式的日期（需晚于出生日期）
4. **计算结果**：点击「计算矫正胎龄」，即可查看包含中间数据的完整结果


## 示例演示
| 输入项         | 示例值         |
|----------------|----------------|
| 出生日期       | 2026-01-01     |
| 出生胎龄       | 34             |
| 目标日期       | 2026-02-01     |

**计算结果**：
目标日期矫正胎龄为 `38W+3`


## 部署到线上（Vercel Hobby计划）
如需将工具部署为公开访问的网页，可通过Vercel免费部署：
1. 将`index.html`文件推送到**GitHub公开仓库**
2. 登录Vercel，点击「Add New → Project」，选择该GitHub仓库
3. 无需配置构建命令（纯静态文件），直接点击「Deploy」
4. 部署完成后，即可通过`xxx.vercel.app`链接访问


## 注意事项
- 日期输入框在老旧浏览器中可能显示为文本框，需手动输入`YYYY-MM-DD`格式
- 出生胎龄的天数部分仅支持`0-6`（符合实际胎龄逻辑）
- 纯本地计算，无数据上传，隐私安全


## 格式说明
| 输入格式 | 对应胎龄   | 转换为天数 |
|----------|------------|------------|
| `34`     | 34周       | 34×7=238天 |
| `34+2`   | 34周2天    | 34×7+2=240天 |
